# Neovim チートシート — 標準操作 + 本設定 (1 枚版)

:::message
このプロジェクトの `init.lua` (setup 章 + neovim-ide 章) を前提にした早見表です。
**印の意味**: 🔧 = 本設定で追加したキーマップ / 無印 = Neovim 標準操作。
**`<leader>` = `Space`**（先頭がスペースのものは全部これ）。困ったら `Esc` 連打で Normal に戻ります。

🔧 キーの**使い方 (操作の流れ) の説明は導入元の章**にあります:
検索・ファイラ・LSP・git 差分 = [neovim-ide 章](neovim-ide.md) / ローカル LLM (`a*`) = [neovim-llm 章](neovim-llm.md)の「使い方」/ Claude Code (`c*`) = [neovim-claudecode 章](neovim-claudecode.md)の「使い方」/ tmux 連携 = [neovim-tmux 章](neovim-tmux.md)
:::

## 1. モード

| やること                    | キー                 |
| --------------------------- | -------------------- |
| 入力開始 (前 / 後)          | `i` / `a`            |
| 行頭 / 行末から入力         | `I` / `A`            |
| 下 / 上に空行を開いて入力   | `o` / `O`            |
| 入力をやめる                | `Esc` (= `Ctrl-[`)   |
| 範囲選択 (文字 / 行 / 矩形) | `v` / `V` / `Ctrl-v` |
| コマンド入力                | `:`                  |

## 2. ジャンプ系検索 / ファジー検索 ★

| やること                       | キー                  | 相当 (VS Code) |
| ------------------------------ | --------------------- | -------------- |
| 🔧 ファイル名で開く            | `<leader>ff`          | `Cmd+p`        |
| 🔧 プロジェクト全文検索        | `<leader>fg`          | `Cmd+Shift+f`  |
| 🔧 開いているバッファ一覧      | `<leader>fb`          | タブ一覧       |
| 🔧 診断 (エラー/警告) 一覧     | `<leader>fd`          | Problems       |
| 🔧 定義へジャンプ              | `gd`                  | F12            |
| 🔧 参照一覧                    | `gr`                  | Shift+F12      |
| 🔧 ホバー (型情報)             | `K`                   | hover          |
| 🔧 次 / 前の診断へ             | `]d` / `[d`           | —              |
| カーソル下の語を検索           | `*` / `#`             | —              |
| 行内ジャンプ (その文字 / 手前) | `f<文字>` / `t<文字>` | —              |
| 対応する括弧へ                 | `%`                   | —              |

:::message
telescope 内: 入力で絞り込み → `Ctrl-j`/`Ctrl-k` で候補移動 → `Enter` で開く → `Esc` で閉じる。
:::

## 3. ファイラ (neo-tree) ★

| やること                             | キー / 操作               |
| ------------------------------------ | ------------------------- |
| 🔧 ファイラ開閉                      | `<leader>e`               |
| ファイラ内: 開く                     | `Enter`                   |
| ファイラ内: 上下移動                 | `j` / `k`                 |
| ファイラ内: 新規ファイル / フォルダ  | `a` (末尾 `/` でフォルダ) |
| ファイラ内: 削除 / リネーム / コピー | `d` / `r` / `c`           |
| ファイラ内: 隠しファイル表示切替     | `H`                       |
| 標準ファイラ (プラグインなし)        | `:Ex` (netrw)             |

## 4. 移動 (Normal)

| キー                | 動き                                 |
| ------------------- | ------------------------------------ |
| `h` `j` `k` `l`     | 左・下・上・右                       |
| `w` / `b` / `e`     | 次語頭 / 前語頭 / 語末               |
| `0` / `^` / `$`     | 行頭 / 最初の非空白 / 行末           |
| `gg` / `G`          | ファイル先頭 / 末尾                  |
| `{` / `}`           | 段落で上 / 下                        |
| `Ctrl-u` / `Ctrl-d` | 半画面 上 / 下                       |
| `5j` `3w`           | 数字 + 動きで倍速 (相対行番号が効く) |

## 5. 編集 — 「動詞 + 対象」

動詞 (`d`削除 / `c`変更 / `y`コピー / `>`インデント) × 対象 (`w`単語 / `iw`単語内 / `ip`段落内 / `i(` `i"`括弧/引用内 / `$`行末まで)。

| 操作                        | キー                          |
| --------------------------- | ----------------------------- |
| 1 文字削除                  | `x`                           |
| 単語 / 行 / 行末まで削除    | `dw` / `dd` / `D`             |
| 単語を打ち直す              | `ciw`                         |
| 括弧 / 引用の中身を打ち直す | `ci(` / `ci"` / `ci{`         |
| 行コピー → 貼付 (下 / 上)   | `yy` → `p` / `P`              |
| 取り消し / やり直し         | `u` / `Ctrl-r`                |
| 直前操作の繰り返し          | `.`                           |
| 矩形選択して先頭一括挿入    | `Ctrl-v` → `I` → 入力 → `Esc` |

## 6. コメントアウト

| やること                | キー                                  |
| ----------------------- | ------------------------------------- |
| 現在行をトグル          | `gcc`                                 |
| 選択範囲をトグル        | Visual 選択 → `gc`                    |
| 範囲指定                | `gc` + 対象 (`gcip` / `gc3j` / `gcG`) |
| ブロックコメント        | `gbc` / `gb` + 対象                   |
| 🔧 VS Code 風一発トグル | `Ctrl-/` (行 / 選択範囲)              |

:::message
コメント記号は filetype 自動判定 (`.ts`→`//` / `.html`→`<!-- -->` / CSS→`/* */`)。
:::

## 7. 検索・置換

| 操作                    | キー                           |
| ----------------------- | ------------------------------ |
| 前方 / 後方検索         | `/語` `Enter` / `?語`          |
| 次 / 前のマッチ         | `n` / `N`                      |
| 🔧 ハイライト消し       | `Esc` (`nohlsearch` 割当)      |
| 全体置換 / 確認しながら | `:%s/旧/新/g` / `:%s/旧/新/gc` |

## 8. LSP 編集支援 (本設定) 🔧

| やること                   | キー                                |
| -------------------------- | ----------------------------------- |
| 定義へ / 参照一覧 / ホバー | `gd` / `gr` / `K`                   |
| リネーム                   | `<leader>rn`                        |
| コードアクション           | `<leader>ca`                        |
| 次 / 前の診断              | `]d` / `[d`                         |
| 補完 (基本は自動表示)      | 入力中に出る / 手動 `Ctrl-x Ctrl-o` |

## 9. ウィンドウ・バッファ

| 操作              | キー                                           |
| ----------------- | ---------------------------------------------- |
| 縦分割 / 横分割   | `Ctrl-w v` (`:vsplit`) / `Ctrl-w s` (`:split`) |
| 分割間を移動      | `Ctrl-w h/j/k/l`                               |
| 分割を閉じる      | `Ctrl-w q`                                     |
| 次 / 前のバッファ | `:bnext` / `:bprevious`                        |
| バッファ一覧      | `:ls` (または 🔧 `<leader>fb`)                 |

## 10. Git 差分 (gitsigns) 🔧

| 操作                        | コマンド                              |
| --------------------------- | ------------------------------------- |
| 行左に追加/変更/削除の記号  | (自動表示)                            |
| 差分プレビュー              | `:Gitsigns preview_hunk`              |
| ステージ / 取消 (hunk 単位) | `:Gitsigns stage_hunk` / `reset_hunk` |

## 11. ファイル・保存・コマンド

| 操作                       | キー / コマンド                 |
| -------------------------- | ------------------------------- |
| 🔧 保存 / 閉じる           | `<leader>w` / `<leader>q`       |
| 保存 / 終了 / 保存して終了 | `:w` / `:q` / `:wq`             |
| 破棄して終了               | `:q!`                           |
| 別ファイルを開く           | `:e パス`                       |
| ヘルプ                     | `:help 語` (例 `:help vim.lsp`) |
| 保存時 prettier 整形       | (自動 / conform)                |
| 対話チュートリアル         | `nvim +Tutor`                   |

## 12. AI 連携 — ローカル LLM (`a*`) / Claude Code (`c*`) 🔧

操作の流れ (開始 → 入力 → 終了) は各章の「使い方」を参照してください:
ローカル LLM は [neovim-llm 章](neovim-llm.md)、Claude Code は [neovim-claudecode 章](neovim-claudecode.md)。

| やること (頭脳)                       | キー                            |
| ------------------------------------- | ------------------------------- |
| AI アクション / チャット (localllm)   | `<leader>aa` / `<leader>ac`     |
| チャット送信                          | Normal `<CR>` (insert 中は `<C-CR>`) |
| チャット内: キー一覧 / 中断 / コードをヤンク | `?` / `q` / `gy`          |
| 選択をインライン処理 (localllm)       | Visual `<leader>ai`             |
| インライン diff の採用 / 破棄         | `gda` / `gdr`                   |
| 選択をチャットへ追加 (localllm)       | Visual `ga`                     |
| FIM ゴースト補完の確定 / 消す         | `Tab` / `Ctrl-e` (insert 中)    |
| Claude トグル / フォーカス            | `<leader>cc` / `<leader>cf`     |
| Claude 再開 / モデル選択              | `<leader>cr` / `<leader>cm`     |
| バッファ / 選択を Claude へ           | `<leader>cb` / Visual `<leader>cs` |
| Claude の diff 承認 / 却下            | `<leader>cy` / `<leader>cn` (または `:w` / `:q`) |

## `<leader>` (Space) 早見

```
Space + w   保存            Space + e    ファイラ開閉
Space + q   閉じる          Space + rn   リネーム
Space + ff  ファイル検索    Space + ca   コードアクション
Space + fg  全文検索        Space + fb   バッファ一覧
Space + fd  診断一覧
Space + a*  ローカル LLM    Space + c*   Claude Code
  (aa アクション / ac チャット)   (cc トグル / cs 送る / cy 承認 / cn 却下)
```

:::message
`which-key.nvim` を入れると `Space` を押した瞬間に上の一覧がポップアップします。
各キーマップに `desc` を付けてあるので、追加設定なしでヒントが出ます。
:::
